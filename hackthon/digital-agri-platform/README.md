# Digital E-Farming and Agri Management System

This is a complete, responsive web platform for managing agricultural activities.

## Features
- **Landing Page**: Modern design with features overview.
- **Authentication**: Register and Login for farmers (simulated with LocalStorage).
- **Dashboard**: Overview of activities, weather widget, and charts.
- **Crop Management**: Add, edit, delete crops with CSV export.
- **Direct Market**: Buy and sell produce simulation.
- **Idea Submission**: Share innovative farming ideas.
- **Query Management**: Ask experts and track query status.
- **Schemes**: View government schemes and support.
- **Admin Panel**: View all registered users, queries, and ideas.

## How to Run
1. Open the project folder.
2. Double-click `index.html` to open it in your web browser.
3. No server installation required (pure HTML/JS).

## Mock Data
The application uses `localStorage` to simulate a database. When you first load the app, some dummy data (crops, products, queries) will be automatically created.
